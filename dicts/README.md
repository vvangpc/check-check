# 📘 Check-Check 核心字典配置指南

`dicts/` 目录下的文件是本工具审查逻辑的“知识库”。通过维护这些文件，您可以让工具更贴合特定技术领域（如机械、通信、生物医药）的审查需求。

## 词典列表及规范

| 文件名 | 作用说明 | 编写格式要求 |
| :--- | :--- | :--- |
| **userdict.txt** | **自定义分词库**。防止专有名词被误切分（如“光刻机”）。 | 每行一个词。例如：`光刻机` |
| **minganci.txt** | **敏感/禁用词拦截**。识别法律红线或不规范用语。 | 每行一个正则规则。例如：`绝对`、`最优` |
| **useless_word.txt** | **冗余词拦截**。识别“大概”、“基本上”等非技术语义词。 | 每行一个规则。例如：`大致` |
| **synonyms.txt** | **同义词映射表**。用于说明书支持验证的实词扩散。 | 同行词用**英文逗号**分隔。例如：`螺钉,螺栓,紧固件` |
| **gudingdapei.txt** | **特征装配关联**。识别部件间的连接关系。 | 使用 `***` 代表占位部件。例如：`***固定在***上` |
| **weizhiguanxi.txt** | **方位空间逻辑**。识别部件间的相对位置。 | 同上。例如：`***位于***内部` |
| **transition_words.txt** | **步骤/流程转接词**。识别动作先后顺序。 | 每行一个词。例如：`随后`、`接着` |
| **reference_signs.txt** | **部件编号档案**。辅助附图标记校验。 | 内部自动维护，建议手动修改前先通过程序界面查看。 |

## 💡 高级技巧

1. **热更新机制**：
   在程序运行期间，您可以通过“设置 -> 配置字典模型”直接编辑并保存。程序会自动调用 `config.load_rules()` 重新加载，无需重启。
2. **正则支持**：
   `minganci.txt` 和 `useless_word.txt` 支持标准正则表达式。例如使用 `[\d]+` 匹配数字。
3. **占位符 `***`**：
   在 `gudingdapei` 和 `weizhiguanxi` 中，`***` 是极其重要的锚点。它告诉算法：这里应该是一个技术部件（名词短语）。

---

> [!IMPORTANT]
> 修改字典文件时，请务必保持 **UTF-8** 编码，否则可能导致中文字符读取乱码。
